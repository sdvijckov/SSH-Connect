# ℹ️ О проекте

## SSH Connect

**SSH Connect** — графический SSH-клиент для подключения к Linux VPS с интеграцией Kitty.

### Возможности

- 🔗 **SSH подключение** с сохранением профилей
- 💻 **Автоматический запуск Kitty** (свёрнутым) после подключения
- 🔍 **Поиск файлов** на сервере с открытием в локальных программах
- 🔄 **Автосохранение** изменений файлов на сервер
- 💾 **Шифрование паролей** в файле сохранений
- ⌨️ **Вставка по Shift+Insert** во все поля

### Технологии

| Компонент | Технология |
|-----------|------------|
| **Язык** | Python 3.10+ |
| **GUI** | Tkinter (ttk, scrolledtext) |
| **SSH библиотека** | paramiko 3.4.0 |
| **SSH терминал** | Kitty (форк PuTTY) |
| **Шифрование** | cryptography 42.0.0 |
| **Хранение данных** | JSON (зашифрованный) |

## Почему Kitty вместо PuTTY?

**Kitty** — это форк PuTTY с расширенными возможностями:

| Возможность | PuTTY | Kitty |
|-------------|-------|-------|
| **Пароль в командной строке** | ❌ | ✅ |
| **Сохранение сессий** | ✅ | ✅ |
| **Встроенный SFTP** | ❌ | ✅ |
| **Скрипты** | ❌ | ✅ |
| **Автоматизация** | ❌ | ✅ |

**Пример запуска Kitty:**
```bash
# Kitty с паролем (свёрнутым)
start /min "Kitty" kitty.exe -ssh root@78.17.11.67 -P 22 -pw password
```
