## 📖 Что такое VNC

**VNC (Virtual Network Computing)** — технология удалённого доступа к графическому рабочему столу.

**В проекте используются:**
- **TightVNC Server** — сервер на VPS (Linux)
- **RealVNC Viewer** — клиент на Windows (встроен в приложение)

**Как работает:**
```
Твой компьютер (Windows)
         ↓
    VNC Viewer
         ↓ (порт 5901)
    Интернет
         ↓ (порт 5901)
    VPS (Linux)
         ↓
    TightVNC Server
         ↓
    XFCE (рабочий стол)
```

## 🛠️ 2. Установка VNC на сервере

### Шаг 1: Подключись к серверу через SSH

Запусти SSH Connect и нажми **"🔗 Подключиться"**

Kitty откроется автоматически — работай в терминале.

---

### Шаг 2: Обнови пакеты

```bash
sudo apt update && sudo apt upgrade -y
```

---

### Шаг 3: Установи VNC сервер и XFCE

```bash
sudo apt install tightvncserver xfce4 -y
```

**Что устанавливается:**
- `tightvncserver` — VNC сервер
- `xfce4` — лёгкая графическая среда (рабочий стол)

---

### Шаг 4: Первый запуск VNC

```bash
vncserver :1
```

**Тебя попросят:**

1. **Ввести пароль VNC** (8 символов максимум)
   - Это пароль для подключения к рабочему столу
   - **Не** пароль от сервера!
   - Запиши его!

2. **Подтвердить пароль** (введи ещё раз)

3. **View-only password?** — введи `n` (не нужно)

---

### Шаг 5: Останови VNC для настройки

```bash
vncserver -kill :1
```

## ⚙️ 3. Настройка сессии

### Шаг 1: Настрой конфиг VNC

```bash
# Резервная копия старого конфига
mv ~/.vnc/xstartup ~/.vnc/xstartup.bak

# Создаём новый конфиг
cat > ~/.vnc/xstartup << 'EOF'
#!/bin/bash
xrdb $HOME/.Xresources
startxfce4 &
EOF

# Делаем исполняемым
chmod +x ~/.vnc/xstartup
```

---

### Шаг 2: Запусти VNC сервер

```bash
vncserver :1 -geometry 1280x800 -depth 24
```

**Параметры:**
- `:1` — номер дисплея (порт 5901)
- `-geometry 1280x800` — разрешение экрана
- `-depth 24` — глубина цвета

---

### Шаг 3: Проверь работу

```bash
# Посмотреть активные сессии
vncserver -list

# Должно быть:
# TigerVNC server sessions:
# X DISPLAY #1
```

## 🚀 4. Запуск через SSH Connect

### Кнопка "🖥️ Открыть GUI"

В приложении **SSH Connect** есть кнопка для быстрого запуска VNC!

**Как использовать:**

1. **Запусти SSH Connect**
   ```bash
   python main.py
   ```

2. **Подключись к серверу**
   - Введи данные (или выбери профиль)
   - Нажми **"🔗 Подключиться"**

3. **Нажми "🖥️ Открыть GUI"**
   - VNC на сервере **перезапустится**
   - VNC Viewer **откроется автоматически**
   - Адрес: `адрес-сервера:5901`

4. **Введи пароль VNC**
   - Пароль который задавал в `vncserver :1`
   - Поставь галочку "Remember password"
   - Нажми OK

**Готово!** 🎉 Ты увидишь рабочий стол XFCE.

---

### Что делает кнопка "🖥️ Открыть GUI"

1. **Перезапускает VNC** на сервере:
   ```bash
   vncserver -kill :1
   vncserver -kill :2
   vncserver :1 -geometry 1280x800 -depth 24
   ```

2. **Запускает VNC Viewer** с адресом сервера

3. **Даёт 2 секунды** на запуск VNC перед открытием Viewer

## 🔧 5. Решение проблем

### Проблема: VNC не запускается

**Решение:**
```bash
# Проверь что установлено
which vncserver

# Если не найдено — установи
sudo apt install tightvncserver -y

# Запусти вручную
vncserver :1
```

---

### Проблема: Серый экран вместо рабочего стола

**Причина:** Неправильный конфиг `xstartup`

**Решение:**
```bash
# Останови VNC
vncserver -kill :1

# Исправь конфиг
cat > ~/.vnc/xstartup << 'EOF'
#!/bin/bash
xrdb $HOME/.Xresources
startxfce4 &
EOF

chmod +x ~/.vnc/xstartup

# Запусти заново
vncserver :1 -geometry 1280x800 -depth 24
```

---

### Проблема: "Too many authentication failures"

**Решение:**
1. В SSH Connect нажми **"🖥️ Открыть GUI"**
2. Скрипт **автоматически перезапустит VNC**

Или вручную:
```bash
# Убить все сессии
vncserver -kill :1
vncserver -kill :2

# Запустить заново
vncserver :1 -geometry 1280x800 -depth 24
```

---

### Проблема: Потерял пароль

**Решение:**
```bash
# Сбрось пароль VNC
rm ~/.vnc/passwd
vncpasswd

# Перезапусти VNC
vncserver -kill :1
vncserver :1 -geometry 1280x800 -depth 24
```

---

### Проблема: VNC не открывается через кнопку

**Причины:**
- `vncviewer.exe` не найден в папке проекта
- VNC сервер не запущен на сервере

**Решение:**
1. Проверь что `vncviewer.exe` в папке проекта
2. Подключись к серверу
3. Запусти VNC вручную: `vncserver :1`
4. Нажми "🖥️ Открыть GUI" снова

## 🔐 Безопасность

### ⚠️ Важно!

1. **VNC пароль ≠ пароль от сервера**
   - Это отдельный пароль
   - Запиши его в надёжное место

2. **Порт 5901 открыт наружу**
   - Рекомендуется использовать SSH-туннель
   - Или настроить фаервол

### Настройка фаервола (рекомендуется)

```bash
# Разрешить подключения только с твоего IP
sudo ufw allow from ТВОЙ_IP to any port 5901
```
