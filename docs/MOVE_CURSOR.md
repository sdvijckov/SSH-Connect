# 📜 Move Cursor — Прокрутка терминала Kitty

## Описание

**Move Cursor** — утилита для удобной прокрутки окна Kitty с помощью клавиатуры.

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Home` | Прокрутка вверх |
| `End` | Прокрутка вниз |
| `Shift + Home` | Прокрутка влево |
| `Shift + End` | Прокрутка вправо |
| `F12` | Пауза / возобновление перехвата |

---

## Автоматический запуск

Move Cursor запускается **автоматически** при нажатии **"🔗 Подключиться"**:

1. Запускается Kitty (свёрнутым)
2. Запускается Move Cursor (свёрнутым)
3. Когда Kitty активно — Page Up/Down прокручивают окно

## Настройка

Все настройки в файле:
```
move_cursor/src/config.py
```

### Основные настройки

```python
# Клавиша паузы: 'f12', 'scroll_lock', 'pause', 'none'
PAUSE_KEY = 'f12'

# Клавиши прокрутки
SCROLL_UP_KEY = 'page_up'
SCROLL_DOWN_KEY = 'page_down'

# Тип прокрутки: 'page' (постранично) или 'line' (по строкам)
SCROLL_TYPE = 'page'

# Количество строк за одно нажатие (если SCROLL_TYPE = 'line')
SCROLL_LINES = 5
```

### Добавление новых терминалов

Чтобы добавить поддержку другого терминала, добавь класс окна в:

```python
WINDOWS_TERMINAL_CLASSES = [
    ...
    "НовыйТерминал",  # Добавь сюда
    ...
]
```

## Решение проблем

### Прокрутка не работает

1. Убедись, что окно Kitty **активно** (в фокусе)
2. Проверь логи в окне Move Cursor
3. Убедись, что move_cursor запущен

### Как проверить что move_cursor работает

В окне move_cursor должны быть логи:
```
12:30:45 - INFO - Move Cursor - Утилита для прокрутки терминалов
12:30:45 - INFO - Бэкенд: WindowsBackend
12:30:45 - INFO - Слушатель клавиатуры запущен
```

### Move Cursor запускается дважды

Закрой лишнее окно move_cursor в панели задач.
