## 📋 Описание

**SSH Connect** — графический SSH-клиент для подключения к Linux VPS с интеграцией:

- **Kitty** (форк PuTTY) — для SSH-терминала
- **VNC Viewer** — для графического рабочего стола

### Основные возможности

- 🔗 **SSH подключение** с сохранением профилей
- 💻 **Автоматический запуск Kitty** (свёрнутым) после подключения
- 🖥️ **Кнопка "Открыть GUI"** — перезапуск VNC и запуск VNC Viewer
- 📁 **Файловый менеджер** — загрузка/скачивание файлов
- 💾 **Шифрование паролей** в файле сохранений
- ⌨️ **Вставка по Shift+Insert** во все поля
