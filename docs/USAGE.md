# 📖 Использование SSH Connect

## 1. Подключение к VPS

### Шаг 1: Ввод данных

При запуске **автоматически загружается последний профиль** (если есть).

| Поле | Значение | Пример |
|------|----------|--------|
| **Профиль** | Выбрать из списка | `adminvps` |
| **Хост** | IP-адрес VPS | `78.10.01.67` |
| **Порт** | Порт SSH | `22` |
| **Пользователь** | Логин | `root` |
| **Пароль** | Пароль от VPS | `******` |
| **SSH ключ** | Путь к ключу (опционально) | `C:\keys\id_rsa.ppk` |

---

### Шаг 2: Подключение

Нажми **"🔗 Подключиться"**

**Успешное подключение:**
- ✅ Статус: "🟢 Подключено"
- ✅ **Kitty запускается автоматически** (свёрнутым)
- ✅ Кнопка "🔗 Подключиться" неактивна
- ✅ Кнопка "🖥️ Открыть GUI" активна

---

### Шаг 3: Работа с Kitty

**Kitty открывается автоматически** после подключения!

- Окно **свёрнуто в панель задач**
- Разверни когда нужно
- **Пароль вводится автоматически**
- Работай в полноценном SSH-терминале

---

## 2. Графический интерфейс (VNC)

### Запуск GUI

1. **Подключись** к серверу
2. **Нажми** "🖥️ Открыть GUI"
3. **VNC Viewer откроется** с адресом сервера

**Что происходит:**
- VNC на сервере **перезапускается**
- VNC Viewer **запускается автоматически**
- Введи пароль VNC (или сохрани в VNC Viewer)

---

### Настройка VNC на сервере

См. [GUI_SETUP.md](GUI_SETUP.md) для подробной инструкции.

**Кратко:**
```bash
# На сервере (через Kitty)
sudo apt update
sudo apt install tightvncserver xfce4 -y
vncserver :1
```

---

## 3. Сохранение профилей

### Сохранить новый профиль

1. Введи данные подключения
2. Нажми **"💾 Сохранить"**
3. Введи имя профиля
4. Профиль сохранится в `ssh_connections.json`

**Файл зашифрован!** Пароли хранятся в зашифрованном виде.

---

### Загрузить профиль

1. Выбери профиль из выпадающего списка
2. Данные **автоматически заполнятся**

---

### Удалить профиль

1. Выбери профиль
2. Нажми **"🗑️ Удалить"**
3. Подтверди удаление

---

## 4. Файловый менеджер

### Просмотр файлов

Справа — список файлов на сервере (`/home/sheldon/`).

- **Двойной клик** → команда `cat filename` (просмотр)
- **"📁 Обновить"** → обновить список
- **"📂 Домой"** → перейти в `/home/sheldon/`

---

### Загрузка файла на сервер

1. Нажми **"⬆️ Загрузить"**
2. Выбери файл на компьютере
3. Файл загрузится в `/home/sheldon/`

---

### Скачать файл с сервера

1. Выбери файл в списке
2. Нажми **"⬇️ Скачать"**
3. Выбери место для сохранения

---

## 5. Быстрые команды

Копирование популярных команд в буфер:

| Кнопка | Команда | Описание |
|--------|---------|----------|
| 📊 uptime | `uptime` | Время работы, нагрузка |
| 💾 df -h | `df -h` | Место на диске |
| 📁 ls -la | `ls -la` | Список файлов |
| 🔝 top | `top -b -n 1` | Процессы |
| 🌡️ free -h | `free -h` | Память |
| 📦 apt update | `sudo apt update` | Обновление пакетов |
| 🐍 python3 | `python3 --version` | Версия Python |
| 📝 tail logs | `tail -f /var/log/syslog` | Логи |

---

## 6. Вставка данных

**Shift+Insert** — вставка из буфера обмена во все поля.

**Контекстное меню** (правый клик) — недоступно.
