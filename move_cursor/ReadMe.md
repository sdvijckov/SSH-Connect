# Move Cursor 🐍

Утилита для удобной прокрутки консольных окон и терминалов с помощью клавиатуры.

## 📖 Описание

Скрипт перехватывает нажатия клавиш `Page Up` / `Page Down` и прокручивает активное окно терминала. Особенно полезен при работе с CLI-агентами, логами серверов и другими инструментами командной строки.

### Особенности

- ✅ **Поддержка всех Windows-терминалов**: cmd, PowerShell, Windows Terminal, Conda, Git Bash, Google Cloud SDK
- ✅ **Горячая клавиша паузы**: `F12` для временного отключения перехвата
- ✅ **Горизонтальная прокрутка**: `Shift + Page Up/Down`
- ✅ **Консольные логи**: Все логи в одном окне, без лишних файлов
- ✅ **Модульная архитектура**: Готово к расширению на Linux/macOS
- ❌ **Без lock-файлов**: Никаких проблем с зависшими процессами


## 📖 Использование

### Запуск

**Важно:** Скрипт работает только при наличии папки `src` рядом с `move_cursor.py`.

```cmd
# Переходите в папку со скриптом
cd D:\Projects python\Help\move_cursor

# Запускаете через python в отдельном окне терминала
python move_cursor.py
```
*Например*
В окне терминала:
D:
cd Projects python\Help\move_cursor
python move_cursor.py
* Запустили скрипт *
В новом окне терминала:
D:
cd Projects python\Help\move_cursor
qwen
* Запустили qwen *

**Что происходит:**
1. Открывается окно терминала с логами скрипта
2. Скрипт перехватывает нажатия `Page Up` / `Page Down`
3. Когда активно окно терминала — прокручивает его
4. В других окнах (браузер, редактор) — не реагирует

**Остановка:** Закройте окно терминала, где запущен скрипт.

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Page Up` | Прокрутка вверх |
| `Page Down` | Прокрутка вниз |
| `Shift + Page Up` | Прокрутка влево |
| `Shift + Page Down` | Прокрутка вправо |
| `F12` | Пауза / возобновление перехвата |

### Поддерживаемые терминалы (Windows)

- ✅ **cmd.exe** — классическая консоль
- ✅ **PowerShell** (все версии, включая 7+)
- ✅ **Windows Terminal** (новый терминал от Microsoft)
- ✅ **Conda Prompt** — для работы с conda-окружениями
- ✅ **Git Bash** — терминал из Git for Windows
- ✅ **Google Cloud SDK Shell** — для работы с GCP
- ✅ **ConEmu, Cmder** — сторонние консоли
- ✅ **VS Code, PyCharm** — встроенные терминалы IDE

## ⚙️ Настройка

Все настройки находятся в файле `src/config.py`. Откройте его и измените параметры:

### Основные настройки

```python
# Клавиша паузы: 'f12', 'scroll_lock', 'pause', 'none'
PAUSE_KEY = 'f12'

# Клавиши прокрутки
SCROLL_UP_KEY = 'page_up'
SCROLL_DOWN_KEY = 'page_down'

# Горизонтальная прокрутка (Shift + клавиша)
HORIZONTAL_SCROLL_ENABLED = True

# Тип прокрутки: 'page' (постранично) или 'line' (по строкам)
SCROLL_TYPE = 'page'

# Количество строк за одно нажатие (если SCROLL_TYPE = 'line')
SCROLL_LINES = 5

# Количество колонок для горизонтальной прокрутки
SCROLL_COLUMNS = 10
```

### Логирование

```python
# Уровень: 'DEBUG', 'INFO', 'WARNING', 'ERROR', 'CRITICAL'
LOG_LEVEL = "INFO"

# Файл лога отключён (логи выводятся в консоль)
LOG_FILE = None
```

**Все логи выводятся в консоль** — в том же окне, где запущен скрипт. Это удобно для отладки и не создаёт лишних файлов.

---

### Настройка скорости прокрутки

**Режим `SCROLL_TYPE = 'page'`** (по умолчанию):
- Прокрутка на целую страницу за одно нажатие
- Быстрая навигация по большим файлам

**Режим `SCROLL_TYPE = 'line'`**:
- Прокрутка на указанное количество строк
- `SCROLL_LINES = 5` — 5 строк за нажатие (рекомендуется)
- Плавная, контролируемая прокрутка

**Пример для медленной прокрутки:**
```python
SCROLL_TYPE = 'line'
SCROLL_LINES = 3  # Всего 3 строки за нажатие
```

**Пример для быстрой прокрутки:**
```python
SCROLL_TYPE = 'line'
SCROLL_LINES = 10  # 10 строк за нажатие
```

## ❓ FAQ

### Скрипт не прокручивает окно

1. Убедитесь, что окно терминала активно (в фокусе)
2. Посмотрите логи в окне запуска — там будет ошибка
3. Убедитесь, что терминал поддерживается (см. список выше)
4. Проверьте, что папка `src` находится рядом с `move_cursor.py`

### Как остановить скрипт?

Просто закройте окно терминала, где запущен скрипт.

### Можно ли изменить клавиши?

Да, откройте `src/config.py` и измените параметры `PAUSE_KEY`, `SCROLL_UP_KEY`, `SCROLL_DOWN_KEY`.

### Скрипт пишет "ImportError: attempted relative import"

Убедитесь, что структура папок сохранена:
```
move_cursor/
├── move_cursor.py
└── src/
    ├── config.py
    ├── main.py
    └── backends/
```
